## Despliegue del Bot

### Para poder interactuar con el chatbot de solicitud de vacaciones, el empleado debe ingresar al sitio web de la empresa:

### https//:www.solucionesdelnorte.com.ar

### Una vez dentro del sitio, deberá seleccionar la opción **"Solicitar Vacaciones Online"**.

### Al acceder a dicha sección, se iniciará la interacción con el chatbot, el cual solicitará los datos necesarios para validar la identidad del empleado:

#### ♦ DNI
#### ♦ Apellido

### Luego de validar la información ingresada, el bot permitirá realizar la solicitud de vacaciones y mostrará la información correspondiente según los datos almacenados en la base de datos.

### Al finalizar el proceso, el chatbot informará al usuario si la solicitud fue aprobada o rechazada. En caso de aprobación, la base de datos se actualizará automáticamente con la información de la nueva solicitud.
